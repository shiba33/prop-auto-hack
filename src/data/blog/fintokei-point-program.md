## はじめに

プロップファームをEAで攻略するなら、チャレンジ合格や出金だけでなく、各社の特典プログラムも活用して収益を最大化したいところです。Fintokeiには2026年1月に導入された **ポイントステージ** （ロイヤリティプログラム）があり、EA運用との相性が非常に良いです。

## Fintokeiポイントステージの仕組み

Fintokei（フィントケイ）のポイントステージは、トレード活動に応じて **XP（経験値ポイント）** が貯まるランクアップ制度です。XPを貯めてティアを上げると、報酬率アップ・損失制限の緩和・無料チャレンジなどの特典が永続的に解放されます。

**XPに有効期限はなく、一度獲得した特典は永久に適用** されます。チャレンジに失敗してもXPは貯まるため、トレードを続ける限りランクは上がり続けます。

## XP獲得方法の全体像

XPは以下の活動で獲得できます。

### チャレンジ・出金で稼ぐXP

| 活動 | 獲得XP |
|------|--------|
| 出金承認 | 300 XP |
| チャレンジ購入（プロトレーダー） | 25〜400 XP |
| チャレンジ購入（スイフトトレーダー） | 25〜200 XP |
| チャレンジ購入（スタートトレーダー） | 10〜100 XP |
| ステップ1 合格 | 50〜250 XP |
| ステップ2 合格 | 75〜150 XP |
| ステップ3 合格 | 150 XP |
| 口座失格 | 50 XP |

### EA稼働で毎日貯まるXP

| 活動 | 獲得XP |
|------|--------|
| デイリーログイン | 10 XP/日 |
| アクティブトレード日 | 10 XP/日 |

### 初回ボーナス（1回のみ）

| 活動 | 獲得XP |
|------|--------|
| 初回アカウント登録 | 200 XP |
| 本人確認（KYC）完了 | 200 XP |
| 初回無料トライアル | 200 XP |

EA運用で最も効率が良いのは **出金** です。出金額に関係なく、1回の出金承認で300XPが入ります。複数口座を運用していれば、口座ごとに出金XPを稼げます。

## 6段階のティアとEA運用に効く特典

ポイントステージは **ティア1〜6の6段階** で構成されています。

### 必要XPと無料チャレンジ

| ティア | 必要XP | 無料チャレンジ |
|--------|--------|----------------|
| ティア1 | 0 | — |
| ティア2 | 2,000 | $5,000 スタートトレーダー |
| ティア3 | 5,000 | $10,000 プロトレーダー |
| ティア4 | 10,000 | $20,000 プロトレーダー |
| ティア5 | 20,000 | $50,000 プロトレーダー |
| ティア6 | 30,000 | $100,000 プロトレーダー |

### 報酬率・損失制限・資本ブースト

| 特典 | ティア2 | ティア3 | ティア4 | ティア5 | ティア6 |
|------|---------|---------|---------|---------|---------|
| 報酬率ブースト | +5% | +10% | — | +15% | +20% |
| 日次損失制限の緩和 | — | +0.5% | +1.0% | +1.5% | +2.0% |
| 最大損失制限の緩和 | — | — | +1.0% | +1.5% | +2.0% |
| 資本ブースト | +5% | +10% | +15% | +20% | +25% |

**報酬率は最大100%まで上昇** します。ティア6では報酬率+20%ブーストが適用され、利益のほぼ全額を受け取ることも可能です。

## EA複数口座運用でのティア6到達シミュレーション

ティア6（30,000XP）はすべての特典が解放される最終目標です。EA運用で現実的なXP獲得ペースをもとに、口座数別の到達月数を計算しました。

### シミュレーションの前提条件

- デイリーログイン: 月300 XP（30日 x 10XP）
- アクティブトレード日: 月220 XP（22日 x 10XP）
- 出金: 口座あたり月1回（300 XP/口座）
- 初回ボーナス: 600 XP（登録+本人確認+トライアル）
- チャレンジ購入・合格XPは含まず（保守的な見積もり）

### 口座数別の到達スピード

| 口座数 | 月間XP | ティア6到達 |
|--------|--------|-------------|
| 1口座 | 約820 | 約36ヶ月（3年） |
| 2口座 | 約1,120 | 約26ヶ月（2年2ヶ月） |
| 4口座 | 約1,720 | 約17ヶ月（1年5ヶ月） |
| 8口座 | 約2,920 | 約10ヶ月 |

※ チャレンジ購入・合格XPを含めると、実際にはさらに短縮されます。

1口座だと約3年かかりますが、4口座で1年半、8口座なら10ヶ月で到達できます。

## EA運用でXPを最大化する3つのコツ

### 出金は回数が命

XPは出金額ではなく **出金回数** に連動します。少額でも出金すれば同じ300XPが入るため、出金可能になったらこまめに出金するのが効率的です。

### チャレンジ失敗もXPに変わる

チャレンジに失敗（失格）しても50XPが付与されます。失敗を恐れてチャレンジしないより、積極的に挑戦した方がXP効率は高くなります。

### 最初の2〜4口座が最もコスパが良い

- 1口座 → 2口座: **到達月数が10ヶ月短縮**
- 4口座 → 8口座: 短縮効果は相対的に小さい

最初の数口座を追加する効果が最も大きいため、無理に8口座を揃える必要はありません。まずは2〜4口座で始めるのが現実的です。

## 速攻プラン vs チャレンジプラン：EA合格率への影響

プランによって合格難易度が大きく異なります。

| プラン | 必要利益 / DD上限 |
|--------|-------------------|
| 速攻プラン | DD上限の **1.67倍** |
| 2STEPチャレンジ | DD上限の **0.8倍** |

速攻プランは2STEPチャレンジの **2倍以上の難易度** です。速攻プランを選ぶ場合は、EA側の合格率が十分に高いことを確認してから挑みましょう。

## おわりに

プロップファームをEAで攻略するなら、トレード以外の収益機会も見逃さないことが大切です。Fintokeiのポイントステージを効率的に進めるなら：

1. **複数口座を同時運用する** — 出金XPが口座数に比例して増える
2. **出金回数を最大化する** — 少額でもこまめに出金してXPを稼ぐ
3. **毎日ログイン・トレードする** — 日常XPも積み重ねると大きい
4. **失敗を恐れずチャレンジする** — 失格でも50XP獲得できる
