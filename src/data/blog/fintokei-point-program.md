## はじめに

プロップファームをEAで攻略するなら、チャレンジ合格や出金だけでなく、各社の特典プログラムも活用して収益を最大化したいところです。Fintokeiにはランクに応じた特典が得られるポイントプログラムがあり、EA運用との相性が良いです。

## Fintokeiのポイントプログラムとは

Fintokei（フィントケイ）には、トレード活動に応じて**XP（経験値ポイント）**が貯まるポイントプログラムがあります。XPを貯めてランクを上げると、実用的な特典が解放されていく仕組みです。

### XPの貯め方

XPは主に以下の活動で獲得できます。

| 活動 | 獲得XP |
|------|--------|
| 出金1回 | 300XP |
| チャレンジ合格 | XP付与あり |
| 日々のトレード活動 | 少量ずつ付与 |

最も効率が良いのは**出金**です。出金額に関係なく、1回の出金で300XPが入ります。

### ランクと特典

ランクは1〜10まであり、上がるごとに特典が充実します。

| ランク | 主な特典 |
|--------|---------|
| 1〜2 | 基本特典（ダッシュボード機能など） |
| 3〜4 | チャレンジ費用の割引 |
| 5〜6 | 利益分配率アップ、追加口座の割引 |
| 7以上 | さらに大きな割引・優遇 |

ランクが上がるほどチャレンジ費用が安くなり、利益分配率も上がるため、**同じEAでも収益性が向上**します。EA運用で複数口座を回すなら、ランクアップの恩恵は大きいです。

## ランク6到達シミュレーション

ランク6は利益分配率アップが得られる実用的な目標です。出金XPのみでランク6に到達するまでの月数を、口座数別に計算しました。

| 口座数 | ランク6到達月数 |
|--------|----------------|
| 1口座 | 46ヶ月（約4年） |
| 4口座 | 12ヶ月 |
| 8口座 | 6ヶ月 |

1口座だけだと約4年かかりますが、8口座を同時運用すれば半年で到達できます。

## XP効率のポイント

### 出金1回 = 300XP

XPは出金回数に連動するため、出金額ではなく**出金回数**が重要です。少額でも出金すれば同じXPが入ります。

### クリスタル（200万円口座）が最もXP効率が高い

口座サイズとXP効率を比較すると、クリスタルプラン（200万円）が最もコスパが良いです。チャレンジ費用に対する出金可能額の比率が高く、結果的にXPあたりのコストが最小になります。

### 口座追加の効果は序盤ほど大きい

- 1口座 → 2口座: **到達月数が半分に**
- 4口座 → 8口座: 短縮効果は相対的に小さい

最初の数口座を追加する効果が最も大きいため、無理に8口座を揃える必要はありません。まずは2〜4口座で始めるのが現実的です。

## Fintokeiの速攻プランとチャレンジプランの難易度比較

プランによって合格難易度が大きく異なります。

| プラン | 必要利益 / DD上限 |
|--------|-------------------|
| 速攻プラン | DD上限の**1.67倍** |
| 2STEPチャレンジ | DD上限の**0.8倍** |

速攻プランは2STEPチャレンジの**2倍以上の難易度**です。速攻プランを選ぶ場合は、EA側の合格率が十分に高いことを確認してから挑みましょう。

## おわりに

プロップファームをEAで攻略するなら、トレード以外の収益機会も見逃さないことが大切です。Fintokeiのポイントプログラムを効率的に進めるなら：

1. **クリスタルプラン（200万円口座）を選ぶ** — XP効率が最も高い
2. **複数口座を同時運用する** — 2〜4口座で到達速度が大幅改善
3. **出金回数を最大化する** — 少額でもこまめに出金してXPを稼ぐ
4. **プラン選びは難易度を確認** — 速攻プランは2STEPの2倍以上の難易度
